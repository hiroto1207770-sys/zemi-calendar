権限とパスワード

合言葉アプリに入るための鍵。全員に配る。変更＝スプレッドシートの拡張機能→Apps Script→プロジェクトの設定→スクリプトプロパティ APP_KEY
管理者パスワード初期値 1270。設定→「🔑管理者メニュー」で入力すると管理機能が開く（アプリを開くたびに要入力）。メニュー内で変更可。忘れた時：スプレッドシートの「meta」シートで key が adminpass の行を削除→1270に戻る
管理者は1人最初にパスで解錠した人（あなた）だけが管理者。引き継ぐときだけ管理者メニューの「登録を解除」→次にパスを入れた人が管理者に
各自の暗証番号本人確認用。本人が初回に設定。忘れた人が出たら：管理者メニューのメンバー一覧で🔓→その人は選び直し＆番号を決め直せる

管理者メニューでできること（設定→🔑）

  • メンバー管理：追加・削除・🎂誕生日（カレンダーに表示）・🔓名前ロック解除
  • AIキー：Google AI Studio の無料キーを入れると「AIにきく」とPDF読み取りが全員使える
  • 目標：水野ゼミ／9期生の目標を入れると全員のホームに表示
  • 管理者パスワードの変更・引き継ぎ

仕組み（どこに何があるか）

アプリ本体GitHub Pages：hiroto1207770-sys/zemi-calendar（無料）
データあなたのGoogleスプレッドシート（projects / items / meta / push シート）。版履歴＝バックアップ（ファイル→変更履歴から復元可）
同期・通知の司令塔同スプレッドシートの Apps Script（GAS）。毎朝のトリガー daily が通知を発射
通知の送信Cloudflare Worker zemi-push（無料）が署名して各端末へ配信

⚠ すべて無料枠。Googleアカウント（スプレッドシート）と Cloudflare のログイン情報は絶対に失くさないこと＝これが全データの鍵。

困ったときの対処

  • 誰かが名前を間違えた：管理者メニュー→その人の🔓→本人が選び直す
  • データを間違って消した：スプレッドシートの「ファイル→変更履歴」で過去の版に戻す
  • 表示がおかしい端末がある：アプリを完全に閉じて開き直す（自動で最新版になる）。ダメならホーム画面のアイコンを入れ直す
  • 通知が来ない人：iOS16.4以上か・ホーム画面のアイコンから開いて🔔を押したか・iPhoneの設定→通知で許可されているかを確認
  • アプリの修正・機能追加：開発担当（Claude）に依頼。配信すると各端末は次に開いたとき自動更新。データは消えない

公開する情報の範囲（把握しておく）

  • 合言葉を知っている人はアプリの共有データ（全体公開の予定など）を見られる。個人情報や秘密はメモ欄に書かない運用に
  • チーム限定・個人指定は「画面上の絞り込み」であり、技術に詳しい人が本気で見れば見えるレベル（ゼミ内の性善説運用）
  • 本当に秘密の内容はこのアプリに載せない
